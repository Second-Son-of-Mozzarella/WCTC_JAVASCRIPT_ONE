Data Attributes

What are they?

Data attributes are attributes that can be added to a HTML element to store some type of value. The syntax for data attributes is quite simple: data-* where the * represents any name that you want to give to the data.

One of the common usages for a data attribute is adding identification to an element so it can be used inside of JavaScript. data-id= "4" would give an element an data attribute named id with a value of 4.

Each element inside of JavaScript has access to the .dataset property which gives access to the data attributes. For example, if I wanted to get the value of the id data attribute, I could access the element and look at the .dataset property which will have the .id property: elementSelector.dataset.id
