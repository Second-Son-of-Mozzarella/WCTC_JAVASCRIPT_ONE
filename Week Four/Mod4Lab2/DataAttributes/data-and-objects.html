Data Attributes and Objects

Using Objects for Attributes

In the JavaScript file you will find a list of students imported from another file. Iterate through the list of students and for each student create a div with the class of “student.” Inside of the div, put an h3 element with the student name inside of it. Add a data attribute named “student-id” to the student div that has the value of the student’s id. Append this div to the innerHTML of the div with the class of “students.” You can use template literal notation to construct a string over multiple lines and with variables: ``.

Data Attributes and e.target

Add an event listener to the div above with the class of “students.” This event listener should be the type of “click.” Upon the event firing, make sure that the target is not the “students” div itself. If it isn’t the “students” div, get the “student” element that was clicked. Please note that if the user clicks a paragraph tag, the paragraph tag will be returned. To get the student div, try using the .closest() method on the target, with an argument of the student class css selector. Alert the Id of the selected student.
