Event Types

There are many different types of events that you can listen for.

Window Load

Add an event listener to the window that alerts “Page Loaded” when complete. You may comment this out after testing it.

Focus and Blur

Add two events to the following input box. One event should trigger on focus, and one event should trigger on blur. On focus, give the input the class of “active.” On blur, remove the class of “active.” For both of the event listeners, use an anonymous function and the keyword this to access and mutate the classList attribute.

Getting the mouse position

Add an event listener to the document that updates the input below on mousemove. A function should get the page position of the mouse from the event object.
