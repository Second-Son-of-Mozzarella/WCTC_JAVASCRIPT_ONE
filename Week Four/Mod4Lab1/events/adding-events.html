Adding Events

There are three different ways to trigger events: HTML Attributes, DOM Handlers, and Event Listeners.

HTML Attribute

Practice

This is what we have been using but is considered bad practice as it is not modular and relies on both HTML code and JavaScript code.

DOM Handler

All modern browser understand this way of creating an event handler, but you can only attach one function to each event handler.

Practice

Add an onclick event to the button element using a DOM handler. When clicked, a function should log “DOM Handler Click.” After that is working, try adding another onclick event to the button element using a DOM handler. When clicked, a seperate function should alert “DOM Handler Click Two.”

Event Listener

Event listeners are a more recent approach to handling events. They can deal with more than one function at a time but they are not supported in older browsers.

Practice

Add an onclick to the button element using an event listener. When clicked, a function should log “Event Listener Click.” After that is working, try adding another onclick event to the button element using an event listener. When clicked, a sperate function should alert “Event Listener Click Two.” From now on, event listeners will be expected in all submissions.
