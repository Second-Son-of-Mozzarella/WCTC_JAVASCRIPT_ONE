Accessing A Node

An element node represents the structure of an HTML element. The terms element and element node are used interchangeably. There are multiple ways to access an element node through the DOM. The following are a few of those ways.

getElementById('id')

Selects an individual element given the value of its id attribute. The HTML element must have an id attribute in order for it to be selectable.

querySelector('css selector')

Selects an individual element given the value of its id attribute. The HTML element must have an id attribute in order for it to be selectable.
