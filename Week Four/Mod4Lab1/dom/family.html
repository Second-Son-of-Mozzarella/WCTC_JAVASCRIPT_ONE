Traversing the DOM

You can traverse the DOM by looking at an element’s parent element (the element that it is in), the element’s child elements (the elements inside of it), and the element’s siblings (the elements that are next to it).

Practice

Create functionality for the following buttons. The buttons should find the current element that has the class of “activeElement.” After finding the active element, the functionality should find the parent, child, or sibling that is appropriate to the button being clicked. You can use the following properties respectively: .parentNode, .firstElementChild, .previousElementSibling, .nextElementSibling.

Please note the use of the word element when getting children and siblings. This gets the element instead of a text node.
