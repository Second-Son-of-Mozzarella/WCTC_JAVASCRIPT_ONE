Object Looping

Looping is an incredibly powerful tool, especially when it comes to a collection of objects. To practice looping, we have an array of students. Each student used a Student constructor and has the following properties:

  • firstName
  • lastName
  • program
  • semester

All students also have the following method which creates an div containing all student information:

  • createElement()



Example Basic Object Looping

You can loop through collections of items by using a for loop. Using a basic for loop, you can loop through each object in the array. Here we have a premade list of students and are looping through each student. By default, this project does not have styling for these items. You will add that next.

Styling Create Styles for the Elements

Sometimes you will create elements purely through code. These elements will not exist until the a script has been executed. You can style for these elements, even if none of them exist yet, by creating styles that will match the output. Any css selectors will work, but it is typical to use class selectors.

The following has four different classes that were assigned in the JavaScript code. Style using class selectors (note the period):

  • .student { /*styles here*/ }
  • .student-name { /*styles here*/ }
  • .student-program { /*styles here*/ }
  • .student-semester { /*styles here*/ }

Add styling for these elements in the myStyles.css file.
