What is this Lab?

This lab is intended to give us more practice when it comes to using for loops. The bulk of this lab will focus on working with existing code that has problems in it. You will debug six exercises, trying to figure out where the problems are occurring.

You will also practice looping with objects, including how to style and sort those objects.

Lastly, this lab will show a few new concepts that will be used in future modules. There are a few exercises around these concepts that are not required, but will give you extra points if completed.

jQuery

jQuery is a JavaScript library that is meant to simplify much of what we do. To use jQuery, you need to include the library in your site, either by having all of the library files on your site, or linking to an online CDN (Content Delivery Network) that hosts the library files. Google hosts jQuery and you will typically see this script added to link to the Google jQuery files.


Link to Google's jQuery CDN 
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>


Example OnClick Listener 
$(document).ready(function(){
    /* jQuery and JavaScript code here */              
});


Lastly, you can use the jQuery library. You can select elements using the jQuery selector syntax $() in conjunction with css selectors.
