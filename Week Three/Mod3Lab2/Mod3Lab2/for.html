For Loops

For loop with a counter and hardcoded test expression

This is and example of a for loop. The for loop has three parts, an initialization expression, a test expression, and an update expression. The counter starts a 0 and runs 10 times from 0 to 9.

For loop with a counter and variable in the test expression

In this for loop, an input is used to get a variable. The variable is used in the text expression to determine how many times the loop has been ran.

For loop with a two variables in the test expression

In this for loop, two inputs are used. One is for the bottom limit test value and the other is used for the top limit test value.

For loop with a variable for the incrementer

This loop has three inputs: one for the base value, one for the top value, and one for the update expression (incrementer).

Looping through an array

For loops can be used to iterate through arrays. In this example, the loop goes through an array of people objects and outputs their name and age.

Looping through an array with a condition

A for loop in conjuction with a conditional can be used to display specific information. For example, using a number input, you can find people who are above a certain age.
