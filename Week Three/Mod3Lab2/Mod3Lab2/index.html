Testing Loops

For Loop

A for loop is a pre-test loop. The loop first tests a condition and then executes a code block if the condition is true.

A for loop has three parts: initialization expression, test expression, and update expression.

Initialization Expression

The initialization expression sets up a variable to use as a counter. This variable is commonly called i. 

let i = 0;

Test Expression

The test expression tests a condition. If the condition is true, the code block of the loop is run. 

i < 5;

Update Expression

The update expression runs after the loop has completed. The loop then returns to the test expression. The update expression is often used to increment the counter value. 

i++;

Example

For loop

for(let i = 0; i < 5; i++)
{
    console.log(i);
}

Output

0
1
2
3
4

While Loop

A while loop is a pre-test loop. The loop first tests a condition and then executes a code block if the condition is true.

A while loop is a simplified for loop. It will continue to run so long as the test condition is true.

Example 1

While loop

let i = 0;
while(i < 5)
{
    console.log(i);
    i++;
}

Output

0
1
2
3
4

Example 2

While loop

let i = 0;
while(i < 0)
{
    console.log(i);
    i++;
}

Do-While Loop

A do-while loop is a post-test loop. The loop first executes a code block and then tests a condition. If the condition is true, it will run the block of code again and test a condition again after the code block is run.

Example 1

Do-while loop

let i = 0;
do
{
    console.log(i);
    i++;
}while(i < 5)

Output

0
1
2
3
4

Example 2

Do-while loop

let i = 0;
do
{
    console.log(i);
    i++;
}while(i < 0)

Output

0
