Try It - To Do List

Using jQuery create a To Do list that stores tasks.

Part 1

Add an event listener for the form that listens for a submit event. Upon submission, a function should do the following:

  • Use the event object to prevent the default functionality
  • Append a new element to div.tasks. This element should be a div with the classes of task and low. Inside it should have an h3 with the task name and a p with the task description
  • Reset the form

Part 2

Create an event listener to div.tasks that finds the closest div.task when clicked. A function should check the closest div.task and see if it has a class of low, high, or done. Based on which class the div has, the following should happen:

  • has low – remove low, add high
  • has high – remove high, add done
  • has done – remove done, add low

Part 3

Create an event listener for the Remove Completed Items button. Upon clicking this button, the button should fade out and every element with the class of done should slide up and be removed upon the completion of sliding up.

Finishing

You must complete this during class and show the instructor or show the instructor your progress at the end of class. This should be the only task being worked on during lab time. Other work can be finished after.
